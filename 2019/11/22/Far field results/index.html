基础

$\mathcal{E}$$(x,y,z;t)$ 表示瞬时电场
$\mathrm{E}$$(x,y,z;t)$ 表示复向量
$\mathcal{E}$$(x,y,z;t)$ = Re[$\mathrm{E}$$(x,y,z;t) e^{jwt}$]

在远场，假设电磁场强度的径向分量为0，且
$\mathrm{E}\theta$=$A(\theta,\phi)\frac{e^{-jkr}}{r}$, $\mathrm{E}\phi$=$B(\theta,\phi)\frac{e^{-jkr}}{r}$
根据麦克斯韦方程组中 $\nabla\times$$\mathrm{E}$ = $-j\omega\mu$$\mathrm{H}$，得
$$
\frac{1}{r^2sin\theta}
\left|
\begin{matrix}
\vec r & r \vec \theta & rsin\theta \vec\phi \
\frac{\partial}{\partial r} & \frac{\partial}{\partial \theta} & \frac{\partial}{\partial \phi} \
0 & r\mathrm{E}\theta & rsin\theta\mathrm{E}\phi
\end{matrix}
\right|
$$
可求得 $-j\omega\mu \mathrm{H}\theta=-\frac{\partial{r\mathrm{E}\phi}}{\partial r}=jk\mathrm{E}\phi$，$\mathrm{H}\theta=-\frac{1}{\eta}\mathrm{E}\phi$
同理，$\mathrm{H}\phi=\frac{1}{\eta}\mathrm{E}_\theta$

方向性系数 (Directivity)

辐射密度(radiation density) $\mathrm{W}$${rad}$ 表示一个周期内的平均功率密度 (可以是近场、远场)。
$\mathrm{W}$${rad}(r,\theta,\phi)$ = $\frac{1}{2}$ Re[$\mathrm{E}$$(r,\theta,\phi)$ $\times$ $\mathrm{H}^*$$(r,\theta,\phi)$]
通过某个封闭曲面的平均功率 $\mathrm{P}_{rad}$ = $\mathop{\iint}\limits_S$$\mathrm{W}$$_{rad}(r,\theta,\phi)$ $\cdot$ $d$$s$

在远场，辐射密度可以表示为
$W_{rad}(r, \theta, \phi) = \frac{1}{2\eta}(|\mathrm{E}\theta|^2 + |\mathrm{E}\phi|^2)$
$= \frac{1}{2\eta}(A(\theta,\phi)^2 + B(\theta,\phi)^2)\frac{1}{r^2}$
通过远场球面的平均功率
$\mathrm{P}_{rad}$ = $\int_0^{2\pi} \int_0^\pi \mathrm{W}_{rad}(\theta,\phi)(r^2sin\theta d\theta d\phi)$
$= \int_0^{2\pi} \int_0^\pi \frac{1}{2\eta}(A(\theta,\phi)^2 + B(\theta,\phi)^2)\frac{1}{r^2} (r^2sin\theta d\theta d\phi)$

辐射强度(radiation intensity) $\mathrm{U}$ 表示在远场，一个周期内，单位立体角的平均功率。
$\mathrm{U}(\theta,\phi)$ = $r^2\mathrm{W}{rad}(r, \theta, \phi)$
$=\frac{1}{2\eta}(|r\mathrm{E}\theta|^2 + |r\mathrm{E}\phi|^2)$
$=\frac{1}{2\eta}(A(\theta,\phi)^2 + B(\theta,\phi)^2)$
与辐射密度相比，辐射强度和$r$无关
平均辐射功率 $\mathrm{P}{rad}$ = $\mathop{\iint}\limits_{\Omega}$ $\mathrm{U}$ $d\Omega$=
$\int_0^{2\pi} \int_0^\pi$ $\mathrm{U}$ $sin\theta d\theta d\phi$
$= \int_0^{2\pi} \int_0^\pi \frac{1}{2\eta}(A(\theta,\phi)^2 + B(\theta,\phi)^2) sin\theta d\theta d\phi$

方向性系数(directivity)是辐射强度的比值，用天线的辐射强度比上各向同性(isotropic)天线的辐射强度。
$\mathrm{D}(\theta,\phi)=\frac{U}{U_0}$ = $\frac{4\pi\mathrm{U}}{\mathrm{P}{rad}}$
$=\frac{4\pi\mathrm{U}}{ \mathop{\iint}\limits{\Omega} \mathrm{U} d\Omega }$

增益 (Gain)

与方向性系数相比，增益会考虑天线的效率。
输入天线的总功率是$P_{in}$, 输入的功率有一部分被反射，一部分被损耗掉(介质损耗、金属损耗), 剩下的才是辐射出去的功率$P_{rad}$。
$P_{rad}=(1-|\Gamma|^2)e_{cd}P_{in}$
其中, $e_{cd}是效率，用来描述金属损耗和介质损耗$

增益有两种定义，一种不考虑反射造成的损耗(Gain)，一种考虑反射造成的损耗(Realized Gain)
Gain的定义是:
$\mathrm{G}(\theta,\phi) = \frac{4\pi\mathrm{U}}{\mathrm{P}{in}(1-|\Gamma|^2)}$
$= \frac{4\pi\mathrm{U}}{\mathrm{P}{rad}}e_{cd} = \mathrm{D}(\theta,\phi)e_{cd}$
Realized Gain的定义是:
$\mathrm{G}(\theta,\phi) = \frac{4\pi\mathrm{U}}{\mathrm{P}_{in}}$

HFSS中的指标

HFSS中的rE

$r\mathrm{E}$ 的定义是 $r\mathrm{E}=r*\mathrm{E}$
HFSS中有多个$r\mathrm{E}$, 分别是

  • rETotal
  • rEPhi
  • rETheta
  • rEX
  • rEY
  • rEZ
  • rELHCP
  • rERHCP
  • rEL3X
  • rEL3Y

其中,
rETheta=$A(\theta,\phi)$, rEPhi=$B(\theta,\phi)$
rETotal = |rETheta|$^2$ + |rEPhi|$^2$

rEX, rEY, rEZ是直角坐标系下的分量:
rEX = rEPhi$sin\phi$ + rETheta$cos\theta cos\phi$, rEY = rEPhi$cos\phi$ + rETheta$cos\theta sin\phi$, rEZ = rETheta*$sin\theta$

rELHCP和rERHCP是左旋圆极化和右旋圆极化:
rETheta = (rETheta - j rEPhi)/2 + (rETheta + j rEPhi)/2
rEPhi = -(rETheta - j rEPhi)/2j + (rETheta + j rEPhi)/2j
rELHCP = (rETheta - j rEPhi)/2 - (rETheta - j rEPhi)/2j = (rETheta - j rEPhi)(1+j)/2
= (rETheta - j rEPhi)/ $\sqrt 2$ $e^{j45}$
rERHCP = (rETheta + j rEPhi)/2 + (rETheta + j rEPhi)/2j = (rETheta + j rEPhi)(1-j)/2
= (rETheta + j rEPhi)/ $\sqrt 2$ $e^{-j45}$

rEL3X和rEL3Y是Ludwig’s third definition of cross polarization?

HFSS中的方向性系数

根据上述分析，方向性系数和$r\mathrm{E}$ 的关系如下:
$\mathrm{DirTheta} = \frac{1}{2\eta}(|\mathrm{rETheta}|^2)\frac{4\pi}{\mathrm{P}{rad}}$
$\mathrm{DirPhi} = \frac{1}{2\eta}(|\mathrm{rEPhi}|^2)\frac{4\pi}{\mathrm{P}{rad}}$
$\mathrm{DirTotal} = \mathrm{DirTheta} + \mathrm{DirPhi} = \frac{1}{2\eta}(|\mathrm{rEPhi}|^2 + |\mathrm{rEPhi}|^2)\frac{4\pi}{\mathrm{P}{rad}}$
$=\frac{1}{2\eta}(|\mathrm{rETotal}|^2)\frac{4\pi}{\mathrm{P}{rad}}$

HFSS中的增益

Gain和$r\mathrm{E}$ 的关系如下:
$\mathrm{GainTheta} = \frac{1}{2\eta}(|\mathrm{rETheta}|^2)\frac{4\pi}{\mathrm{P}{rad}(1-|\Gamma|^2)}$
$\mathrm{GainPhi} = \frac{1}{2\eta}(|\mathrm{rEPhi}|^2)\frac{4\pi}{\mathrm{P}{rad}(1-|\Gamma|^2)}$
$\mathrm{GainTotal} = \mathrm{DirTheta} + \mathrm{DirPhi} = \frac{1}{2\eta}(|\mathrm{rEPhi}|^2 + |\mathrm{rEPhi}|^2)\frac{4\pi}{\mathrm{P}{rad}(1-|\Gamma|^2)}$
$=\frac{1}{2\eta}(|\mathrm{rETotal}|^2)\frac{4\pi}{\mathrm{P}{rad}(1-|\Gamma|^2)}$

Realized Gain和$r\mathrm{E}$ 的关系如下:
$\mathrm{Realized GainTheta} = \frac{1}{2\eta}(|\mathrm{rETheta}|^2)\frac{4\pi}{\mathrm{P}{in}}$
$\mathrm{Realized GainPhi} = \frac{1}{2\eta}(|\mathrm{rEPhi}|^2)\frac{4\pi}{\mathrm{P}{in}}$
$\mathrm{Realized GainTotal} = \mathrm{DirTheta} + \mathrm{DirPhi} = \frac{1}{2\eta}(|\mathrm{rEPhi}|^2 + |\mathrm{rEPhi}|^2)\frac{4\pi}{\mathrm{P}{in}}$
$=\frac{1}{2\eta}(|\mathrm{rETotal}|^2)\frac{4\pi}{\mathrm{P}{in}}$

轴比 (Axial Ratio)

将二维直角坐标系中一点 $(x,y)$, 乘以如下矩阵，得到新的点 $(x’,y’)$. 新的点是原来的点逆时针旋转 $\theta$得到的.
$$ \left[
\begin{matrix}
cos \theta & -sin \theta \
sin \theta & cos\theta
\end{matrix}
\right]
$$

轴比的定义是椭圆极化的长轴和短轴的复振幅之比.

Polarization Ratio

两种极化方式正交的电场的复振幅之比，如LHCP/RHCP, Theta/Phi
